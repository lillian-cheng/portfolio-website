MUSE Music App

Role:

  • UI design
  • Wireframing
  • Prototyping

Duration:

Sept-October 2020

Tools:

Adobe XD, Photoshop, Pen & Paper

Overview

MUSE was the very first mobile design I created about 1 week into my UI certification program. From start to finish it took a total of 5 weeks.

Ideology

I picked a music app as my first project because I’m familiar with this type of application. I love music and I’ve been a user of Spotify & SoundCloud for years.

A lot of music player apps tend to use a solid color (like black or white) for the background and overall theme, but I would love to create a more vibrant and colorful interface for users with different visual preferences.

Key Functionality for MUSE

  • Add, delete a song from a playlist - Data Management
  • Providing the user song suggestions (recently listened, now trending, suggested etc.) when editing their playlist
  • A convenient way to delete a song
  • Search a song through the name, artist or keywords - Gathering Input
  • When user access the search function, a search bar and search suggestions appear
  • Play, pause, skip to the next song - Navigation
  • While a song is playing, allow users to play, pause, repeat, skip or go back to last song
  • On this page, include other functions like return to the home page, add to playlist, share etc.
